# Blood Calculator

Скрипт оценивает, за сколько дней можно набрать нужное число донаций для получения званий "Почетный донор Москвы" и "Почетный донор России", выбирая тип донации так, чтобы достичь цели как можно быстрее.

## Результат

Moscow: 1 years, 1 months, 16 days (406 days)
    0 blood, 30 plasma donations
Russia: 2 years, 3 months, 16 days (826 days)
    0 blood, 60 plasma donations

## Что учитывается

- два типа донаций: цельная кровь и плазма.
- интервалы между донациями: кровь→кровь 60 дней, кровь→плазма 30 дней, плазма→кровь 14 дней, плазма→плазма 14 дней.
- алгоритм выбирает следующую донацию для минимального срока достижения критериев.

## Критерии получения

- Почетный донор России (госуслуги): <https://www.gosuslugi.ru/help/faq/donor/102288>
- Почетный донор Москвы (mos.ru): <https://www.mos.ru/news/item/96243073/>
- Интервалы между донациями (donor.blood.ru): <https://donor.blood.ru/how-donor/frequency-donations.php>
